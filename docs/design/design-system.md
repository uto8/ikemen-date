## タイポグラフィ

### フォントファミリー

| 役割 | フォント | 備考 |
|---|---|---|
| 見出し（日本語） | `Noto Sans JP` | weight: 700 |
| 本文（日本語） | `Noto Sans JP` | weight: 400 |
| 英数字 | `Inter` | weight: 400 / 600 |
| モノスペース | 未定 | コード表示が必要な場合のみ |

Google Fonts 読み込み例:
```
Noto+Sans+JP:wght@400;700
Inter:wght@400;600
```

### フォントサイズスケール

| トークン | px | rem | 用途 |
|---|---|---|---|
| `text-xs` | 12px | 0.75rem | タグ・バッジ・タイムスタンプ |
| `text-sm` | 14px | 0.875rem | 補足文・ラベル・キャプション |
| `text-base` | 16px | 1rem | **本文（基準）** |
| `text-lg` | 18px | 1.125rem | カード内の名前・小見出し |
| `text-xl` | 20px | 1.25rem | 未定 |
| `text-2xl` | 24px | 1.5rem | セクション見出し |
| `text-3xl` | 32px | 2rem | ページタイトル |
| `text-5xl` | 48px | 3rem | ヒーローコピー（トップページのみ） |

### 行間（line-height）

| トークン | 値 | 用途 |
|---|---|---|
| `leading-tight` | 1.25 | 見出し・カードタイトル |
| `leading-normal` | 1.5 | UI ラベル・ボタン |
| `leading-relaxed` | 1.7 | **本文（基準）** |
| `leading-loose` | 2.0 | 未定 |

### 字間（letter-spacing）

| トークン | 値 | 用途 |
|---|---|---|
| `tracking-tight` | `-0.01em` | 大きな見出し（詰め気味） |
| `tracking-normal` | `0` | 本文 |
| `tracking-wide` | `0.05em` | 未定 |

## シャドウ

| トークン | 値 | 用途 |
|---|---|---|
| `shadow-xs` | `0 1px 2px rgba(0,0,0,0.04)` | 入力フィールド・フォーム部品 |
| `shadow-sm` | `0 2px 8px rgba(0,0,0,0.06)` | **カード（メイン）** |
| `shadow-md` | `0 4px 16px rgba(0,0,0,0.10)` | ドロップダウン・ポップオーバー |
| `shadow-lg` | `0 8px 32px rgba(0,0,0,0.14)` | モーダル・ボトムシート |
| `shadow-none` | `none` | フラットな要素 |

> 基本的に `shadow-sm` だけで完結させる。多段影は使わない。

## ブレークポイント

**モバイルファースト**で記述する（デフォルトが 375px 基準）。

| トークン | 最小幅 | 対象デバイス |
|---|---|---|
| （デフォルト） | 375px | **iPhone SE・スマホ（メイン）** |
| `sm` | 640px | 大きめスマホ・縦向きタブレット |
| `md` | 768px | タブレット横向き |
| `lg` | 1024px | 小型ノートPC |
| `xl` | 1280px | デスクトップ |
| `2xl` | 未定 | 未定（大型モニター対応が必要な場合） |

> 実装優先度: デフォルト（375px）→ `sm` → `md` → `lg` 以上は未定

## Tailwind 設定への反映メモ

```js
// tailwind.config.ts（抜粋）
theme: {
  extend: {
    colors: {
      primary: {
        50:  '#e0f7fa',
        100: '#b2ebf2',
        500: '#00c1d4',
        600: '#00a3b3',
        700: '#007f8c',
      },
    },
    fontFamily: {
      sans: ['Noto Sans JP', 'Inter', 'sans-serif'],
    },
    borderRadius: {
      '2xl': '16px',
    },
    boxShadow: {
      sm: '0 2px 8px rgba(0,0,0,0.06)',
      md: '0 4px 16px rgba(0,0,0,0.10)',
      lg: '0 8px 32px rgba(0,0,0,0.14)',
    },
  },
},
```
